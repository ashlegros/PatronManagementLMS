# PatronManagementLMS

Name: Ashbel Legros

Course: CEN 4333C

Program Objective: This LMS is intended for a librarian who needs to manage patrons. This LMS will allow them to add patrons, remove patrons by ID, and display all current patrons. The initial startup will ask for a file path to a text file. This text file is formatted specifically in a id-name-address-overdueFindAmount format using the dashes (-) as separators. Once a file has been leaded, a on screen menu will be loaded and allow for the user to perform the actions as mentioned above.

Note: You can download the testData.txt file as a starter for the program. It has three patrons already formatted.
